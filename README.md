# JobMatrix 🚀

JobMatrix is a premium, all-in-one platform designed for job seekers and students preparing for high-paying technical roles. Built with a focus on modern aesthetics, it features a cutting-edge **Dark Glassmorphic design** and a highly functional interface.

## ✨ Key Features

- **💼 Job Portal**: A dedicated space to explore high-paying tech roles with real-time mock listings, including salary and location details.
- **📚 Preparation Hub**: Comprehensive study materials for:
  - **Aptitude**: Quantitative, Logical, and Verbal reasoning.
  - **Technical**: In-depth notes on OS, DBMS, Computer Networks, and System Design.
  - **HR Round**: Behavioral interview tips and common question bank.
- **🏢 Company Guides**: Detailed breakdowns of the hiring processes for top tech giants like Google, Amazon, Microsoft, and Netflix.
- **📝 DSA Master Sheet**: A topic-wise Data Structures and Algorithms tracker (inspired by SDE sheets) to master interview problems.
- **🌓 Dark/Light Mode**: Seamless theme switching with persistent state across sessions.
- **🎨 Premium UI**: Glassmorphism effects, Bento-grid layouts, and smooth micro-interactions.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for better SEO and accessibility.
- **Vanilla CSS**: Custom design system using CSS Variables, Flexbox, and Grid.
- **JavaScript (ES6)**: Modular logic for dynamic rendering and view management.
- **Lucide Icons**: Premium vector icons for a modern look.

## 🚀 Getting Started

Since this is a high-performance Vanilla JS application, no installation is required.

### Local Development
1. Clone or download this folder.
2. Open `index.html` in any modern web browser.
3. **Recommended**: Run with a local server for the best experience (especially for local storage and icon loading).
   - Using Python: `python -m http.server 8000`
   - Using Node: `npx serve .`

## 📂 File Structure

- `index.html`: Main entry point and application structure.
- `style.css`: Design system, glassmorphic utilities, and theme variables.
- `main.js`: Application logic, theme management, and dynamic rendering.
- `data.js`: Centralized mock data for jobs, companies, and DSA problems.
- `README.md`: Project documentation.
